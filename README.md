# awesome-learn-driving

### Safe Driving Makes Happy Life

**[線上體驗 Demo](https://yuritsaitw.github.io/awesome-learn-driving/)**

## 為什麼做這個？

我是一個想學開車的上班族。學開車的原因很單純 — 想帶家裡的狗狗貓貓上山下海。

但我是那種比較容易緊張的人。光想到「高速公路上突然爆胎怎麼辦」就覺得腿軟。駕訓班能教會我過 S 彎、路邊停車，但那些真正讓人手足無措的突發狀況 — 煞車突然失靈、濃霧中能見度剩不到 30 公尺、被後車追撞 — 這些不可能在練習場上模擬。

另外，筆試的題庫雖然有標準答案，但光靠死背很難記住「為什麼要這樣做」。

所以我做了這個：**一個讓你在安全的螢幕前，把各種嚇人的駕駛情境都先走過一遍的互動訓練工具。**

不只是讀規則，而是讓你實際做決策、看後果、操作模擬駕駛座 — 等到真的上路，至少腦袋裡已經演練過該怎麼應對了。

## 這個工具能幫你什麼？

- **容易緊張的新手** — 先在螢幕上經歷突發狀況，降低真實遭遇時的恐慌感
- **準備筆試的考生** — 不只背答案，透過情境理解「為什麼這樣做才對」
- **想複習的老手** — 你上次遇到爆胎是什麼時候？再走一遍流程不吃虧

## 功能特色

### 七階段學習循環

每個情境都包含完整的學習迴圈，從故事代入到實際操作：

| 階段     | 你會做什麼                                                  |
| -------- | ----------------------------------------------------------- |
| **情境** | 跟著故事角色進入緊急狀況 — 有時間、地點、車速，緊張感拉滿   |
| **決策** | 三個選項，只有一個是對的 — 另外兩個看起來也很合理           |
| **後果** | 動畫告訴你選了之後會發生什麼事                              |
| **教學** | 一步一步教你正確的處置流程，每步都有圖解                    |
| **測驗** | 4 題選擇題，確認你真的記住了                                |
| **模擬** | 第一人稱駕駛座！轉方向盤、踩煞車、按雙黃燈 — 用正確順序操作 |
| **結果** | 看看你拿了幾分，附上關鍵步驟摘要方便複習                    |

### 六大緊急情境

| 情境         | 你會學到                                         |
| ------------ | ------------------------------------------------ |
| 高速公路拋錨 | 雙黃燈 → 移至路肩 → 放三角架 → 撥打救援          |
| 行駛中爆胎   | 握穩方向盤 → 鬆油門 → 雙黃燈 → 靠路肩 → 撥打救援 |
| 大雨濃霧     | 霧燈 → 減速 → 雙黃燈 → 安全停靠                  |
| 輕微追撞事故 | 雙黃燈 → 拍照存證 → 移車 → 報警/保險             |
| 煞車失靈     | 連踩煞車 → 降檔 → 雙黃燈 → 安全處 → 手煞車       |
| 窄路會車     | 煞車 → 靠右 → 閃大燈 → 鳴喇叭示意                |

### 駕駛模擬器

沉浸式第一人稱駕駛座視角，所有控制元件皆可互動：

- **方向盤** — 拖曳轉向
- **煞車 / 油門踏板** — 點擊操作
- **儀表板按鈕** — 雙黃燈、霧燈、喇叭、雨刷
- **排檔桿** — 切換檔位
- **情境專屬按鈕** — 三角架、電話等
- **鍵盤快捷鍵** — 支援鍵盤操作

### 其他特色

- **角色驅動** — 每個情境由不同台灣角色帶入（年齡、職業、地區都不同），讓你更容易代入
- **純 SVG 繪圖** — 所有插圖皆為手工編寫的 SVG，無圖片資源、即時載入、完全響應式
- **複習模式** — 完成的情境可以直接跳到任一階段複習，不用重頭來過

## 技術架構

- **Build**: Vite + React 18 + TypeScript
- **Styling**: Tailwind CSS v4
- **Linting**: ESLint + Prettier + Husky pre-commit hooks
- **Deploy**: GitHub Actions → GitHub Pages

## 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建置生產版本
npm run build

# 程式碼檢查
npm run lint

# 格式化
npm run format
```
