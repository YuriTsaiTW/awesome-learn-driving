# awesome-learn-driving

台灣駕駛緊急應變互動訓練

**[Demo](https://yuritsaitw.github.io/awesome-learn-driving/)**

## 為什麼做這個？

大多數駕駛人熟悉交通規則，但在真正遭遇緊急狀況時，往往會因為恐慌而做出錯誤判斷。這個專案的核心理念是：**透過情境式體驗學習，讓駕駛人在安全的環境中預先演練緊急應變流程**，縮短「知道規則」與「能正確反應」之間的落差。

每個情境都由一位台灣在地角色（不同年齡、職業、地區）帶入故事，讓使用者產生代入感，並在互動中記住正確的處置步驟。

## 功能特色

### 七階段學習循環

每個情境都包含完整的學習迴圈：

| 階段     | 說明                                                   |
| -------- | ------------------------------------------------------ |
| **情境** | 故事敘事 — 角色、時間、地點，營造緊迫感                |
| **決策** | 三選一 — 一個正確、兩個看似合理但錯誤的選項            |
| **後果** | 動畫呈現你的選擇帶來的結果                             |
| **教學** | 圖文並茂的正確處置步驟                                 |
| **測驗** | 4 題選擇題，驗證學習成效                               |
| **模擬** | 第一人稱駕駛座模擬器，實際操作方向盤、踏板、儀表板按鈕 |
| **結果** | 成績與評級，附帶關鍵步驟摘要                           |

### 六大緊急情境

| 情境         | 核心技能                                         |
| ------------ | ------------------------------------------------ |
| 高速公路拋錨 | 雙黃燈 → 移至路肩 → 放三角架 → 撥打救援          |
| 行駛中爆胎   | 握穩方向盤 → 鬆油門 → 雙黃燈 → 靠路肩 → 撥打救援 |
| 大雨濃霧     | 霧燈 → 減速 → 雙黃燈 → 安全停靠                  |
| 輕微追撞事故 | 雙黃燈 → 拍照存證 → 移車 → 報警/保險             |
| 煞車失靈     | 連踩煞車 → 降檔 → 雙黃燈 → 安全處 → 手煞車       |
| 窄路會車     | 煞車 → 靠右 → 閃大燈 → 鳴喇叭示意                |

### 駕駛模擬器

沉浸式第一人稱駕駛座視角，所有控制元件皆可互動：

- **方向盤** — 拖曳轉向
- **煞車 / 油門踏板** — 點擊操作
- **儀表板按鈕** — 雙黃燈、霧燈、喇叭、雨刷
- **排檔桿** — 切換檔位
- **情境專屬按鈕** — 三角架、電話等
- **鍵盤快捷鍵** — 支援鍵盤操作

### 其他特色

- **角色驅動** — 每個情境由不同台灣角色帶入，增加共感
- **純 SVG 繪圖** — 所有插圖皆為手工編寫的 SVG，無圖片資源、即時載入、完全響應式
- **複習模式** — 完成的情境可透過階段選擇器直接跳到任一階段複習

## 技術架構

- **Build**: Vite + React 18 + TypeScript
- **Styling**: Tailwind CSS v4
- **Linting**: ESLint + Prettier + Husky pre-commit hooks
- **Deploy**: GitHub Actions → GitHub Pages

## 本地開發

```bash
# 安裝依賴
npm install

# 啟動開發伺服器
npm run dev

# 建置生產版本
npm run build

# 程式碼檢查
npm run lint

# 格式化
npm run format
```
